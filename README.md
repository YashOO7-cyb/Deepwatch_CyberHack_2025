# 🔍 Deepwatch: Darkweb CSAM Tracker

## 📌 Overview

**Deepwatch** is an interactive web-based dashboard designed to monitor, analyze, and track CSAM (Child Sexual Abuse Material) activities across the dark web. The platform provides real-time insights, OSINT investigation support, and blockchain logging for transparency and accountability.

---

## 🚀 Features

* 📊 **Interactive Dashboard**

  * Visual analytics using charts and graphs
* 🧭 **Sidebar Navigation**

  * Home
  * Detected CSAM Cases
  * Real-time Monitoring
  * OSINT Investigations
  * Blockchain Logs
* 📈 **Data Visualization**

  * Line graph for CSAM case trends
  * Pie chart for accuracy & performance metrics
* 🔍 **OSINT Integration**

  * Support for open-source intelligence investigations
* 🔗 **Blockchain Logs**

  * Secure and traceable activity logging
* ⚙️ **Settings & Account Management**

---

## 🛠️ Tech Stack

* **Frontend:** React.js (Vite / Create React App)
* **Styling:** Tailwind CSS
* **Charts:** Chart.js & react-chartjs-2
* **Icons:** Lucide React

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/deepwatch-dashboard.git
cd deepwatch-dashboard
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Run the Project

```bash
npm run dev
```

### 4️⃣ Open in Browser

```
http://localhost:5173
```
